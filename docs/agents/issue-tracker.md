# Issue Tracker

Issues for this repository live in GitHub Issues for `lukebax/meeting-minutes-workflow`.

Engineering skills that create or triage issues should use the GitHub repository configured as `origin`:

```text
git@github.com:lukebax/meeting-minutes-workflow.git
```

Do not use generated workflow outputs, meeting recordings, transcripts, `.venv`, or `.DS_Store` files as issue attachments unless the user explicitly confirms that the material is safe to share.
